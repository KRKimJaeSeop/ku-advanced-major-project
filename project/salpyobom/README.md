## ✅ 구현 기능

### WebView 앱 연동
안드로이드 네이티브 앱 내에 WebView를 적용하여 웹 기반 서비스를 앱에서 구동할 수 있도록 구현하였다.

### 반응형 레이아웃
모바일(세로/가로)과 PC 환경 모두에서 화면 요소가 적절히 배치되도록 반응형 레이아웃을 적용하였다.

### 회원가입 → 로그인 플로우
신규 사용자가 회원가입 후 로그인까지 이어지는 인증 플로우를 구현하였다.

### 독거노인 명단 리스트
임시 데이터를 기반으로 복지사가 담당하는 독거노인 명단을 리스트 형태로 확인할 수 있다.

### 상세 정보 화면
명단에서 특정 어르신을 선택하면 해당 어르신의 상세 정보 화면으로 이동한다.
